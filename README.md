# 猪尾巴小游戏

[![微信公众号-雪之梦技术驿站-brightgreen.svg](https://img.shields.io/badge/%E5%8D%9A%E5%AE%A2%E5%9B%AD-%E6%96%AD%E4%BA%86%E7%9A%84%E5%BC%A6-blueviolet)](https://www.cnblogs.com/duanledexian/p/15430572.html)


## 运行环境

- jdk 8.0

## 编译方法

下载main.jar后，在命令行输入

```
java -jar main.jar
```

## 使用方法

可选择在线对战、人人对战、人机对战

在线对战可以选择创建对局或者加入对局
